# Projeto Acadêmico — NoSQL com Neo4j

Este projeto acadêmico foi desenvolvido na disciplina de **Banco de Dados NoSQL**
do curso de **Análise e Desenvolvimento de Sistemas (ADS)**.

O objetivo do projeto é aplicar conceitos de **bancos de dados orientados a grafos**
utilizando o **Neo4j**, explorando a **modelagem de nós e relacionamentos**, a
**importação de dados no formato JSON** por meio da biblioteca **APOC** e a
realização de **consultas analíticas com a linguagem Cypher**.

A partir dos dados importados, foram realizadas análises relacionadas ao uso de
**hashtags em tweets**, explorando os relacionamentos entre **usuários**, **tweets**
e **hashtags** em um banco de dados orientado a grafos.

Os scripts disponibilizados neste repositório representam a **implementação técnica**
do projeto e possuem finalidade **exclusivamente acadêmica e de aprendizado**.
Os arquivos JSON utilizados na importação **não estão versionados** neste repositório.
